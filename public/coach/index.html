1. 프로그램 개요

STRIDE 제품을 승마장 현장에서 가장 잘 추천할 수 있는 사람은 코치님입니다. 마주/회원과 매일 만나고, 말의 상태를 가장 잘 아는 사람이니까요.

이 프로그램의 핵심: 코치님은 "추천만" 하면 됩니다. 재고를 사서 쌓아둘 필요 없고, 배송을 신경쓸 필요도 없습니다. 추천 코드 하나로 커미션이 자동으로 발생합니다.

승마장 정기배송 (구독 모델)

코치가 한 번 세팅하면, 매월 자동으로 배송 + 매월 자동으로 커미션이 발생하는 구독 모델입니다.

구분내용
신청코치가 "우리 승마장 말 8마리, DAILY 정기배송 신청"
배송매월 자동 배송 → 승마장 도착
결제각 마주에게 개별 청구 (또는 승마장 일괄 결제 후 마주에게 정산)
코치 수익한 번 세팅 → 매월 커미션 자동 발생 (패시브 수익)
예시: 말 8마리 × DAILY 35만원/통 × 2.5개월 교체 = 월 ~112만원 매출 → 코치 커미션 월 ~11만원 (자동)

왜 이 구조가 좋은가

✅ 코치 입장

  • 재고 부담 없음 (사서 쌓아둘 필요 X)
  • 배송/결제/세금 신경 안 써도 됨
  • 추천만 하면 커미션 자동 발생
  • 정기배송 세팅 시 매월 패시브 수익
  • 다른 코치 소개 시 추가 수익

✅ STRIDE 입장

  • 고객 데이터 직접 확보 (마주 연락처)
  • 재주문/구독 유도 가능
  • 코치가 마케팅 역할 → 광고비 절감
  • 가격 통제 가능 (코치 마진 없음)
  • 품질/배송 직접 관리 → 브랜드 신뢰

마스터 코치 소개 수당 (2단까지만)

마스터 코치가 다른 코치를 SCA로 소개하면, 그 코치의 판매에서 5%가 마스터 코치에게 추가 지급됩니다. 2단(마스터→파트너)까지만 적용되며, 그 이상은 없습니다.

⚠️ 다단계(MLM)가 아닌 이유:
• 수익 원천: 하위 모집이 아닌 제품 판매
• 레벨: 2단(마스터→파트너)까지만, 무한 확장 없음
• 가입비: 없음 (초기 구매 강제 없음)
• 재고 부담: 없음 (STRIDE 직배송)
• 법적 분류: 제휴 마케팅 (Affiliate) — 쿠팡 파트너스, 아마존 어소시에이츠와 동일 구조
