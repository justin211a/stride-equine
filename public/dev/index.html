B. 제품 인사이트 (가장 중요)

인사이트상세STRIDE 반영
커큐민(강황) 핵심이카인 아메리카에는 커큐민 있음, 까발로에는 없음. 코치님 본인도 커큐민 직접 복용 (어깨 통증 효과 봄). 두 가지 같이 먹는다고 함FLEX에 커큐민 2,000mg 포함 → 경쟁 우위
기호성 차이까발로: 10마리 중 2마리 거부. 이카인: 전부 잘 먹음. 커큐민 유무가 기호성에도 영향 가능3종 기호성 시스템 설계 (당밀+사과+호로파)
관절 메커니즘인대/힘줄 염증 → 관절 전이 → 관절 기능↓ → 뼈 충격 증가. 링본(Ring Bone) = 뼈 증식. 트리트먼트(관절 주사) 1~2년 1회. 영양제 = 주사 줄이려고 먹이는 것"주사 1회 > 영양제 수개월" 세일즈 메시지
개똥쑥+노니코치님이 직접 먹이고 본인도 복용. 개똥쑥: 산통(colic) 예방 + 장내 염증 제거. 노니: 통풍/항염 (코치님 통풍 환자). 여름 물 급하게 먹으면 장내 독소 → 다리로 가서 병 발생 → 개똥쑥이 예방GUT 제품에 개똥쑥 추가 검토 (Dr. Schloss 확인 필요)
발굽 크림현재 보습+코팅 제품 있음. 코치님: "살균까지 되면 1석 3조"STRIDE Hoof Coat 제품 기획
MSM 거부MSM = 쓴맛 → 말이 사료 거부. 코치님 확인MSM 제외, L-메티오닌+콘드로이틴으로 대체

C. 비즈니스 인사이트

항목내용
장제사 네트워크코치님이 장제사 모임에 유일하게 참석하는 코치. 모임도 코치님 가게에서 진행. 장제사 = 여러 마장 순회 → 영양제 추천 채널
간식/샴푸 확장말 간식: 볼 형태, 당근/사과맛, 원가 극히 저렴 (알파파 40kg 3.5만원). 100개 소량부터 시작 가능. 샴푸: 코치님 레시피 있음
리워드 포인트코치님 제안: 커미션을 포인트 → 소모품 교환 구조. 코치들이 소모품 구매에 돈 쓰는데, 그걸 STRIDE 포인트로 대체
이카인 마케팅대회 생중계에서 선수 이름 뜰 때 브랜드 로고 같이 노출 (광고비 지불). 선수에게 브랜드 재킷 제공 → 착용하고 경기
까발로 vs 이카인까발로 = 전 세계 1위, 마케팅 별로 안 함 (이미 잘 팔리니까). 이카인 = 최근 급성장, 돈 많이 써서 홍보. 코치님은 원래 까발로 → 이카인으로 전환
코치 영향력"강 감독이 알아서 해" — 코치님 승마장 마주들은 코치님이 추천하면 100% 따름
