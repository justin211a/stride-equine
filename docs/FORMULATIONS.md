# STRIDE 제품 성분표 전체 레퍼런스

## 공통: 기호성 시스템 (전 제품 필수 적용)

| 기호성 원료 | 기능 | 배합량 범위 |
|---|---|---|
| 당밀 분말 (Dried Molasses) | 단맛 베이스, 사료 접착 | 2,000~3,000mg |
| 건조 사과 분말 (Dried Apple) | 향미 + 단맛 | 2,000~2,500mg |
| 호로파 (Fenugreek) | 마스킹 + 기호성 | 1,000~1,500mg |

> ⚠️ MSM은 의도적으로 전 제품에서 배제. 이유: 극도로 쓴맛 → 말이 사료 거부. 대체: L-메티오닌 + 콘드로이틴 황산.
